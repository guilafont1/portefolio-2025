# Images du Portfolio

## Photo de profil

Placez votre photo de profil dans ce dossier avec le nom : **`profile.jpg`**

### Spécifications recommandées :
- Format : JPG ou PNG
- Taille : 400x400 pixels minimum (carré)
- Poids : < 500 KB
- Fond : Fond uni ou transparent de préférence

### Nom du fichier
Le fichier doit s'appeler exactement : `profile.jpg`

Si vous utilisez un autre format (PNG), modifiez le nom dans le template `templates/home/index.html.twig` ligne ~75.
